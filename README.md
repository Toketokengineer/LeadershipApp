# リーダーシップ行動（PWA）

日付ごとに「今日、自分が何をしたか」をカレンダーから選んでサッと記録できるPWAです。iPhoneで「ホームに追加」するとアプリのように使えます。

## 使い方

1. このフォルダをローカルサーバーで配信する（例: `npx serve .` や `python3 -m http.server`）
2. iPhoneのSafariで開く
3. **共有** → **「ホーム画面に追加」** でホームに追加
4. 追加したアイコンから起動するとフルスクリーンで利用可能

## 機能

- **今週のカレンダー**: 月〜日を丸い日付で表示。今日がハイライト、タップで日付を選択
- **日付ごとの記録**: 選択した日付に「リーダーシップ行動」の自由入力と「AI提案」から選んだ内容を保存
- **振り返り・再編集**: 過去の日付をタップすれば、その日の記録をいつでも確認・編集可能
- **データ保存**: ブラウザの localStorage に日付（YYYY-MM-DD）をキーにして保存
- **オフライン対応**: 一度開けばオフラインでも利用可能
- **iOS対応**: ステータスバー・safe area・ホーム画面追加に対応

## アイコン（任意）

PWAのインストール時にアイコンを表示したい場合は、次のファイルを `icons/` に置いてください。

- `icon-192.png` … 192×192px
- `icon-512.png` … 512×512px

`icons/icon.svg` をベースに画像編集ソフトやオンライン変換でPNGに変換できます。未設定でもアプリは利用可能です。

## ファイル構成

```
├── index.html      # メイン画面（カレンダー・記録フォーム）
├── manifest.json   # PWAマニフェスト
├── sw.js           # サービスワーカー（キャッシュ）
├── styles.css      # スタイル
├── app.js          # カレンダー・日付別記録・localStorage
├── icons/
│   ├── icon.svg    # アイコン素材（SVG）
│   ├── icon-192.png  # 任意
│   └── icon-512.png  # 任意
└── README.md
```

## ローカルで試す

```bash
npx serve .
# または
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080`（または表示されたURL）を開いてください。iPhoneで試す場合は、同じWi‑Fi内で「あなたのPCのIP:8080」でアクセスできます。
